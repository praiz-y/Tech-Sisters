TECH SISTERS 
📌 Project Overview
-------------------

The site presents a decentralized hub concept where women can learn,
connect, showcase talent, and access career opportunities in tech.

This project was created as a frontend practice project to improve
HTML, CSS, and general website layout skills.


🌐 Website Sections
-------------------
- Home
- About Us
- Services
- Contact Us

Main content includes:
- Hero section introducing Tech Sisters
- Community mission and vision
- Services offered
- Social media connection section
- Footer with quick navigation links


✨ Features
-----------
- Clean landing page layout
- Responsive structure (basic)
- Organized content sections
- Simple navigation menu
- Informational service cards
- Community-focused design concept


🛠 Technologies Used
--------------------

- CSS3

(No frameworks used — built for practice purposes)
